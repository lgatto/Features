Anatomy of a SummarizedExperiment object

Mass spectrometry-based quantitative proteomics data can be representated as a matrix of quantitative values for features (PSMs, peptides, proteins) arranged along the rows, measured for a set of samples, arranged along the columns. SummarizedExperiment class (Morgan et al. 2019) handle this kind of tabular data.

These are the component pieces of the SummarizedExperiment for data representation.

Anatomy of a QFeatures object

The QFeatures package (Gatto and Vanderaa 2020) is based on the SummarizedExperiment and MultiAssayExperiment classes and provides infrastructure to manage and analyse quantitative features from mass spectrometry experiments. It follows a hierarchical structure: spectra compose peptides which in turn compose proteins. The main advantage of this structure is that is very easy to navigate across spectra, peptide and protein quantitative data.

library("QFeatures")
cptac <- readQFeatures(___, ecol = ___, sep = "\t", name = "peptides", fnames = "Sequence")

We also use the optional arguments peptides to name assay in the QFeatures object and fnames to set the row-names of thepeptides assay to the peptide sequences.

We can also extract the list of data within the object by using the assays function.

Now use the class function on the output of the assays function.

assays(cptac)
    assays(cptac) %>% 
  class()

SimpleList

The SimpleList class is an S4Vectors representation of Lists with metadata. The length of this SimpleList is 1, since there is only one assay in this dataset (but there could be more in experiments such as single cell proteomics) .

Filtering out contaminants

filterFeatures function to filter QFeatures contaminants

Use the filterFeatures function to filter contaminants in cptac

library("magrittr")
cptac_filter <- cptac %>%
    filterFeatures(~ Reverse == "") %>%
    filterFeatures(~ Potential.contaminant == "")
library("magrittr")
cptac_filter <- cptac %>%
    filterFeatures(~ Reverse == "") %>%
    filterFeatures(~ Potential.contaminant == "")

Compare number of rows of cptac which is 11466 with number of rows of cptac_filter which is 11357
